What is an Armstrong Number?

An Armstrong number is a number that is equal to the sum of its own digits raised to the power of the number of digits it has. For example, let's take the number 153. It has 3 digits, so we raise each digit to the power of 3 (the number of digits) and then sum them up:

1^3 + 5^3 + 3^3 = 1 + 125 + 27 = 153

As you can see, the sum of the cubes of its digits equals the number itself, making 153 an Armstrong number.

How To Calculate Armstrong Number

An Armstrong number (also known as a narcissistic number or plenary number) is a number that is equal to the sum of its own digits each raised to the power of the number of digits.

In other words, let's say we have a number "abcd...n" where "a", "b", "c", ..., "n" are its digits (in base 10).
The number is considered an Armstrong number if: a^n + b^n + c^n + ... + n^n = abcd...n

Let's see another example, 153 is an Armstrong number because: 1^3 + 5^3 + 3^3 = 1 + 125 + 27 = 153
The number 153 is equal to the sum of its own digits (1, 5, and 3), each raised to the power of 3, which is the total number of digits in the number.
